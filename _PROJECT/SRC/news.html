Видеоинструкция
для серии KÜBELE DE018

Компания GuteWetter выпустила новую серию душевых ограждений KÜBELE. Для облегчения процесса сборки мы выпустили видеоинструкцию для душевого ограждения формы четверть круга, ведь наши покупатели проживают в разных уголках страны и не у каждого есть возможность заказать фирменный монтаж, многие собирают душевые ограждения самостоятельно.

Используя монтажную инструкцию, прилагаемую к каждому изделию, вы сможете избежать нарушений технологии сборки. Правильная сборка душевого ограждения – залог его долгой службы!

Посмотреть видеоинструкцию вы можете на нашем канале.

Видеоинструкции для других форм душевых ограждений будут опубликованы по мере их создания. Подписывайтесь на наш канал, чтобы быть в курсе всех наших новинок.
